# 🎯 Full-Stack Portfolio — Screaming Architecture

## 🇺🇸 English

This is my portfolio repository. Here you can find some info about me, the technologies I use, and ways to communicate with me.

It is built using **Next.js 16**, **Tailwind CSS**, and **Screaming Architecture** for maximum scalability and maintainability.

### ✨ Features

- ✅ **Screaming Architecture** - The folder structure clearly screams its purpose.
- ✅ **Modular Components** - Each feature is independent and self-contained.
- ✅ **shadcn/ui Components** - Clean, professional, and accessible UI.
- ✅ **Dark/Light Mode** - Dynamic theme toggling.
- ✅ **Responsive Design** - Works perfectly on mobile, tablet, and desktop.
- ✅ **Fully Typed** - TypeScript for strict type safety.
- ✅ **Optimized** - High-speed development using Turbopack.

## 📁 Project Structure / Estructura del Proyecto

```
minimalist-portfolio/
├── 📂 app/ # Next.js App Router (Root layout & pages)
├── 🎯 features/ # Screaming Architecture Features
│ ├── about/ # About Section
│ ├── resume/ # Resume Section
│ ├── portfolio/ # Portfolio Projects Section
│ ├── contact/ # Contact Form Section
│ └── shared/ # Shared feature components (ThemeProvider, etc.)
├── 🎨 components/
│ └── ui/ # shadcn/ui primitive components
├── 📚 lib/ # Local data & portfolio-data.ts
└── 📦 public/ # Static assets (images, icons)
```

## 🚀 Getting Started / Guía de Inicio

```bash
# Clone the repository / Clonar el repositorio
git clone <repository-url>

# Go to project directory / Entrar al directorio
cd project-reorganization

# Install dependencies / Instalar dependencias
pnpm install

# Run development server / Iniciar servidor local
pnpm dev
```
